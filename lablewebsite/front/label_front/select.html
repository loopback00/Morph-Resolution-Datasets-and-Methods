背景
在网络直播带货的过程中，某些商品可能涉及敏感词汇或被平台限制词汇，直播内容可能会受到法规或平台规则的限制，为了避免平台的自动审查系统对某些敏感或限制性词汇的过滤，同时也为了确保信息能顺畅传达到每一位观众，主播必须在言辞之间找到平衡，主播和商家经常诉诸于一种特殊的语言策略：使用变体词。变体词就是网络语言作为一种不规范语言的显著特色，人们往往出于避免审查、表达情感、讽刺、娱乐等需求将相对严肃、规范、敏感的词用相对不规范、不敏感的词来代替，用来代替原来词的新词就叫做变体词。这些变体词虽然在听觉上或视觉上与原词相似，但却能够巧妙地避开自动审查系统的识别,不触碰到平台的红线,这种做法有时也被称为“打擦边球”。

变体词类型
结构变体词汇 : 变体词汇：某诊某所 原词：诊所 ;      变体词汇：临某床 原词：临床 ;       变体词汇：肠胃 原词：小肠小胃 ;
音形变体词汇 : 变体词汇：k糖 原词：抗糖 ;      变体词汇：k老 原词：抗老 ;
语义变体词汇 : 变体词汇：白褂褂/白衣服的人 原词：医生 ;      变体词汇：火火症 原词：炎症 ;       变体词汇：红色管道 原词：红色管道 ;
